## Installation

### Install SASS and SCSS Lint
```
gem install sass scss_lint
```

### Install node and npm
```
brew install node
```

### Install grunt - task runner
```
npm install -g grunt-cli
```

### Install dependencies
```
cd /path/to/html-starter && npm install
```

## Usage

### Start a server
```
npm run server
```

### Build scss and js files
```
grunt build
```

### Watch files change
```
grunt watch
```

### Check css style
```
npm run scss-lint
```
